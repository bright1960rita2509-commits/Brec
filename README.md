Breccia Escrow Global
Breccia Escrow Global is a secure, escrow-based African trade marketplace designed to build trust in cross-border and local trade. It addresses the "trust gap" in African commerce by providing a robust financial infrastructure that holds funds until milestones are met and verified.
🚀 Features
Escrow Infrastructure: Secure holding of funds until trade milestones are fulfilled.
Role-Based Access: Specialized dashboards for Buyers, Sellers, and Admins.
Trade Lifecycle: Automated state machine for Creating, Funding, Shipping, Inspecting, and Releasing/Disputing trades.
Security First: JWT-based authentication, KYC/AML integration capabilities, and audit logging.
🛠 Tech Stack
Frontend: React, TailwindCSS, React Router
Backend: Node.js/Express, TypeScript
Database: PostgreSQL managed via Prisma
⚙️ Project Structure
/src: Frontend source code (components, pages, services, hooks)
/server.ts: Backend entry point and API routes
/prisma: Database schema and migrations
/docs: Additional documentation
🚀 Getting Started
Install dependencies: npm install
Setup environment: Configure .env with required secrets (see .env.example).
Run in development: npm run dev
